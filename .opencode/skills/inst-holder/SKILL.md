# 机构持股 & 股东数据查询 Skill

通过 `tq.get_financial_data_by_date()` 查询机构持股明细和股东汇总数据。

## 初始化

```python
import sys
sys.path.insert(0, 'C:/new_tdx64/PYPlugins/user')
from tqcenter import tq
tq.initialize(__file__)
```

## 可用脚本

| 脚本 | 功能 |
|------|------|
| `query_inst_holder.py` | 查询机构持股（机构/基金/QFII/社保/私募） |
| `query_top10_holder.py` | 查询前十大股东合计数据 |

使用时修改脚本顶部的 `CODE` 变量为目标股票代码，`periods` 为要查询的报告期。

## 数据完整性说明（⚠️ 重要）

A 股披露规则导致 FN 字段在不同报告期统计范围不同，**直接跨期对比会得出错误结论**：

| 报告期 | 对应年月日 | 披露范围 | 数据完整性 |
|--------|-----------|---------|-----------|
| **年报（A）** | 1231 | 全部股东明细 | ✅ 完整 |
| **中报（H1）** | 0630 | 全部股东明细 | ✅ 完整 |
| **第一季报（Q1）** | 0331 | 仅前十大股东 | ⚠️ 不完整 |
| **第三季报（Q3）** | 0930 | 仅前十大股东 | ⚠️ 不完整 |

**后果**：同一只股票，年报显示 500 家基金持有、季报显示只有 10 家——这不代表基金卖了，而是季报只披露前十大重仓。

**正确做法**：
1. ❌ **不要** 直接对比 Q1 vs H1 vs Q3 vs A 的家数/持股量
2. ✅ 同类型对比：A vs A、H1 vs H1、Q1 vs Q1
3. ✅ 或用中报/年报的完整数据做趋势分析
