# Python 脚本功能列表

| 序号 | 文件名 | 功能说明 |
|-----|--------|----------|
| 1 | `analyze_data.py` | 分析上证指数近3个月数据，包括价格统计、月度表现、成交量分析、波动性分析等 |
| 2 | `build_ashare_history_db.py` | 通过TdxQuant API构建A股历史数据库，支持批量下载股票数据并保存到CSV和SQLite |
| 3 | `create_zt_sector.py` | 读取涨停股票列表，创建自定义板块"今日涨停"并发送股票到该板块 |
| 4 | `download_zt_data.py` | 下载涨停股票的历史数据（支持指定日期范围），保存为CSV文件到cvs目录 |
| 5 | `import_tdx_vipdoc_to_sqlite.py` | 从通达信vipdoc目录的.day文件导入数据到SQLite数据库，支持按日期范围筛选 |
| 6 | `list_all_zt.py` | 检测并列出所有涨停股票，获取其代码、名称、现价、涨停价 |
| 7 | `list_sector_stocks.py` | 列出指定板块的成分股，支持ZTB、QS、BUY、ZZT等板块 |
| 8 | `list_user_sectors.py` | 获取用户自定义板块列表 |
| 9 | `list_zt_stocks.py` | 列出涨停板块(ZTB)的股票 |
| 10 | `potential_ztb.py` | 筛选潜在涨停股票，基于成交量放大、均线排列、动量等技术指标 |
| 11 | `query_a_share_db.py` | 查询A股数据库，支持按代码、日期范围筛选，输出数据库统计信息 |
| 12 | `query_vanke.py` | 查询万科(000002.SZ)股票数据 |
| 13 | `query_stock_to_csv.py` | 查询指定股票数据并导出为CSV文件，支持命令行参数指定代码、日期范围、数据库路径 |
| 14 | `store_a_share_data.py` | 通过TdxQuant API获取近一个月A股数据并存储到SQLite数据库 |
| 15 | `update_zt_sector.py` | 读取涨停股票列表，发送股票更新到ZT板块 |
| 16 | `refresh_sectors.py` | 刷新板块缓存，获取用户自定义板块列表 |
| 17 | `ztb.py` | 筛选今日涨停板股票，获取近3个月历史数据保存到ztb目录 |
| 18 | `ztb_analysis.py` | 对涨停板股票进行技术分析，计算均线、成交量比率、波动率等指标 |
| 19 | `fix_a_share_full_code_suffix.py` | 修复SQLite数据库中股票代码后缀，将无后缀代码添加.SH/.SZ后缀 |
| 20 | `list_tables.py` | 列出SQLite数据库中的所有表 |
| 21 | `convert_to_md.py` | 将HTML文件转换为Markdown格式 |
| 22 | `test_api.py` | 测试TdxQuant API的get_market_data接口 |
| 23 | `test_api2.py` | 测试TdxQuant API不指定field_list的情况 |
| 24 | `test_block.py` | 测试不同block_type获取板块股票列表 |
| 25 | `test_send2.py` | 测试发送股票到自定义板块功能 |
| 26 | `test_send_block.py` | 测试创建新板块并发送股票 |
| 27 | `tdxdata_test.py` | 获取上证指数日线数据并保存为CSV |
| 28 | `tdxdemo.py` | TdxQuant API使用示例，涵盖K线、财务、板块、预警等各类接口 |
| 29 | `scrape_tdx.py` | 爬取通达信量化帮助文档网站页面 |
| 30 | `scrape_tdx_smart.py` | 智能版爬虫，支持限流控制和重试机制 |
| 31 | `tqcenter.py` | TdxQuant API Python接口封装（核心库文件） |

## 数据库

| 数据库文件 | 说明 |
|-----------|------|
| `a_share_full.db` | 完整历史数据（约8962只股票，2350万条记录） |
| `a_share_monthly.db` | 近一个月数据 |

## 目录结构

```
.
├── TdxQuant/           # TdxQuant API文档
├── tdx_quant_md/       # 转换后的Markdown文档
├── tdx_quant_site/    # 爬取的HTML文档
├── data/              # 下载的股票数据
├── ztb/                # 涨停板股票历史数据
├── cvs/               # CSV格式历史数据
└── *.db               # SQLite数据库
```

## 常用命令

```bash
# 更新数据库（从vipdoc文件）
.venv\Scripts\python.exe import_tdx_vipdoc_to_sqlite.py --last-month

# 查询数据库统计
.venv\Scripts\python.exe -c "import sqlite3; conn = sqlite3.connect('a_share_full.db'); cur = conn.cursor(); cur.execute('SELECT MIN(date), MAX(date), COUNT(*) FROM stock_data'); print(cur.fetchone()); conn.close()"
```
