# 调用通达信公式

TdxQuant支持调用通达信公式系统进行计算。

## 新增函数（2026-01-31）

- formula_format_data：格式化K线数据
- formula_set_data：向通达信公式系统设置数据
- formula_set_data_info：向通达信公式系统设置数据信息
- formula_get_data：获取公式中的设置数据
- formula_zb：调用通达信技术指标公式
- formula_xg：调用通达信条件选股公式
- formula_exp：调用通达信专家系统公式

## 批量调用函数（2026-02-07）

- formula_process_mul_xg：批量调用选股公式
- formula_process_mul_zb：批量调用指标公式

## 使用示例

```python
# 调用技术指标公式
result = tq.formula_zb(
    formula_name='MA',
    formula_arg='5,10,20',
    stock_code='000001.SZ',
    stock_period='1d',
    count=10
)

# 批量调用指标公式
result = tq.formula_process_mul_zb(
    formula_name='MACD',
    formula_arg='',
    xsflag=2,
    return_count=1,
    stock_list=['000001.SZ', '600000.SH'],
    stock_period='1d',
    count=-1
)
```

**注意**：调用通达信公式返回值字段名由"Data"改为"Value"（2026-03-27更新）
