# 交易函数

TdxQuant提供完整的交易功能，支持模拟交易和实盘交易。

## 账户相关函数

- stock_account：获取资金账户句柄
- query_stock_asset：账户资产查询
- query_stock_orders：查询账户委托信息
- query_stock_positions：查询账户持仓信息

## 交易执行函数

- order_stock：交易执行函数
- cancel_order_stock：撤单（2026-03-27新增）

## 信用交易

order_stock支持信用交易：
- 担保品买入
- 担保品卖出
- 融资买入
- 融券卖出

## order_type类型

| 名称 | 值 | 说明 |
|------|-----|------|
| STOCK_BUY | 0 | 买 |
| STOCK_SELL | 1 | 卖 |
| CREDIT_BUY | 0 | 担保品买入 |
| CREDIT_SELL | 1 | 担保品卖出 |
| CREDIT_FIN_BUY | 69 | 融资买入 |
| CREDIT_SLO_SELL | 70 | 融券卖出 |

## price_type类型

| 名称 | 值 | 说明 |
|------|-----|------|
| PRICE_MY | 0 | 自填价 |
| PRICE_SJ | 1 | 市价 |
| PRICE_ZTJ | 2 | 涨停价/笼子上限 |
| PRICE_DTJ | 3 | 跌停价/笼子下限 |

## Status类型

| 名称 | 值 | 说明 |
|------|-----|------|
| WTSTATUS_NULL | 0 | 无效单 |
| WTSTATUS_NOCJ | 1 | 未成交 |
| WTSTATUS_PARTCJ | 2 | 部分成交 |
| WTSTATUS_ALLCJ | 3 | 全部成交 |
| WTSTATUS_BCBC | 4 | 部分成交部分撤单 |
| WTSTATUS_ALLCD | 5 | 全部撤单 |

## 使用示例

```python
# 买入股票
result = tq.order_stock(
    stock_code='000001.SZ',
    order_type=0,  # 买入
    price_type=0,  # 自填价
    price=10.5,
    quantity=100
)
```
